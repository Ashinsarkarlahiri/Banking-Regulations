# Banking-Regulations: Recent Global Regulatory Changes (EU, UK, US)
## Overview

This repository contains a comprehensive research-based analysis of the recent changes in global banking regulations, with a strong focus on the Basel III End-Game (Basel 3.1) framework.
The project covers capital, liquidity, stress testing, ICAAP/ILAAP, provisioning, and jurisdiction-wise divergence across Europe, United Kingdom, and the United States.

This project is aligned with modern regulatory expectations in quant risk, banking supervision, capital modelling, and regulatory policy analysis.

## Key Objectives

- Summarize major regulatory developments across Basel, EU, UK, and US regulatory bodies.

- Explain changes in capital requirements across risk types (credit, market, CCR, CVA, operational).

- Compare implementation timelines, calibration, and jurisdictional differences.

- Highlight practical implications for banks, quant teams, and risk management groups.

- Provide a structured, technical, and recruiter-friendly reference document for regulatory analytics.
